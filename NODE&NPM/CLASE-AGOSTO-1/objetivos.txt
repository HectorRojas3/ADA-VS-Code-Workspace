JS UTILIZA PROMESAS Y CALLBACKS

PROMESAS
EN JS UNA PROMESA ES UN OBJETO QUE SIRVE PARA REPRESENTAR VALORES QUE PUEDEN ESTAR DISPONIBLES AHORA,
EN EL FUTURO O NUNCA. 

CALLBACKS
SON FUNCIONES QUE SE LLAMAN CUANDO SE COMPLETA UNA TAREA ASÍNCRONA

TAREA ASÍNCRONA
PUEDE EJECUTARSE ESTA Y LUEGO CONTINUAR CON LA EJECUCIÓN DEL PROGRAMA
CUANDO DICHA TAREA ESTÉ COMPLETA

POR LO TANTO UNA FUNCIÓN ASÍNCRONA ES LA QUE SE EJECUTA SIN TENER LUGAR EN UN PROCESO O LA CAUSA QUE LO PRODUCE

    1. PERMITEN REALIZAR TAREAS SIN BLOQUEAR LA EJECUCIÓN DEL PROGRAMA
    2. PERMITEN QUE EL PROGRAMA CONTINUE MIENTRAS LA TAREA SE REALIZA EN SEGUNDO PLANO
    3. SON UTILES PARA TRABAJAR CON OPERACIONES QUE PUEDEN TARDAR EN COMPLETARSE:
        a. REALIZAR CALCULOS COMPLEJOS
        b. CARGAR DATOS DESDE UNA BASE REMOTA

ASYNC, AWAIT

CUANDO SE LLAMA UNA FUNCION async, ESTA DEVUELVE UN ELEMENTO promise
ESTE ELEMENTO PROMISE DEBE RESOLVERSE CUANDO LA FUNCION DEVUELVE UN VALOR. 

LA FUNCION async PUEDE CONTENER UNA EXPRESION (palabras reservadas) await 
ESTE OPERADOR ES USADO PARA ESPERAR UNA promise. Y SOLO PUEDE USARSE DENTRO DE 
LA FUNCION async
